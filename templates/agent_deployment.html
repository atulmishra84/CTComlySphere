Quick-Start Guide
Docker (recommended)
# Pull and run the collector agent docker run -d \ --name complysphere-agent \ --restart unless-stopped \ -e PLATFORM_URL=https://your-platform-url \ -e API_TOKEN=<paste-token-here> \ -e SCAN_INTERVAL=60 \ -v /var/run/docker.sock:/var/run/docker.sock \ complysphere/collector-agent:latest
One-Line Install (Linux / macOS)
# Download and run the install script curl -fsSL https://your-platform-url/static/agent/install.sh \ | bash -s -- \ --token <paste-token-here> \ --platform https://your-platform-url
Docker Compose
# docker-compose.yml version: "3.8" services: complysphere-agent: image: complysphere/collector-agent:latest restart: unless-stopped environment: PLATFORM_URL: https://your-platform-url API_TOKEN: <paste-token-here> SCAN_INTERVAL: "60" volumes: - /var/run/docker.sock:/var/run/docker.sock
